

## 题目描述
你的任务相当简单。只要用不超过 $n$ ($n \le 100$) 根火柴摆出一个尽量大的，且能被 $m$ ($m \le 3000$) 整除的正整数即可。

火柴摆放对应数字消耗：
- `1`: 2根, `2`: 5根, `3`: 5根, `4`: 4根, `5`: 5根
- `6`: 6根, `7`: 3根, `8`: 7根, `9`: 6根, `0`: 6根

### 输入格式
输入包含 $T$ 组样例。每组样例包含 2 个正整数 $n$ 和 $m$。

### 输出格式
输出共 $T$ 行，每行输出按照要求可以摆出的数的最大值；若无解则输出 -1。
